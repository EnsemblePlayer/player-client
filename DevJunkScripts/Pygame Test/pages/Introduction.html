Introduction

This is a Pygame-1.6 tutorial for windows users about sound. It covers the very basic stuff and hopefully it will make you start using sound in game developing with pygame. When I first opened the pygame documentation, I could not understand a thing. This tutorial is an attempt to make the pygame documentation more readable (to me of course). Also I am not some kind of expert in sound or programming. I just happen to like music and python very much. To read this tutorial, basic python knowledge is required.

This tutorial is made with the Mozilla composer and the Nvu editor. So it is best viewed with the Mozilla browsers. Mind my English language as I am not a native speaker.
